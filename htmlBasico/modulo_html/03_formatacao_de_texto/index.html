Formatação de texto

A tag "b" coloca a palavra ou o texto em negrito

A tag "strong" tb coloca o texto ou palavra em negrito, mas dá mais relevância

A tag "i" coloca o texto ou palavra em itálico

A tag "u" coloca o texto ou palavra em sublinhado

A tag "s" coloca o texto ou palavra em tachado

O HTML é uma linguagem de marcação, então não usa-lo como estético, e sim o css para efeito de estética
